JavaScript - Fonctions

Objectif

Le but de cet exercice est de créer une fonction:

  • La fonction doit être capable d'afficher dans la console les multiples d'un nombre donné inférieurs à une limite donnée. (c'est donc une fonction à deux paramètres).
  • Tester la fonction au moins une fois, soit dans le code JavaScript soit en l'appelant depuis la console.
