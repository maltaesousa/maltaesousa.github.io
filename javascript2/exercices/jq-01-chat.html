Bienvenue

Le but de cet exercice est de prendre en main jQuery. A gauche se trouve un panneau "CHAT" dans lequel vous devrez afficher des messages.

Les messages se trouvent dans un objet dans le fichier JavaScript joint à cette page.

Exercice 1

Affichez les messages dans le chat en respectant les règles suivantes:

  • Les messages sont justifiés à gauche
  • Vos messages (vous êtes le user "me") sont justifiés à droite
  • Respectez la couleur choisie par l'utilisateur en colorant le texte du message

Pour vous aider, un exemple en HTML a été mis plus bas: servez-vous en comme gabarit et utilisez ces classes CSS de Bootstrap (par exemple "col-8", "offset-4" ou "right") elles vous seront utiles pour respecter les consignes. Vous devrez ajouter les couleurs du texte vous-même.

Ajoutez des "id" dans les tags HTML dès que vous en avez besoin!

Exercice 2

Faites en sorte que l'envoi de messages fonctionne:

  • Quand vous cliquez sur "Envoyer!" un message est ajouté dans le CHAT.
  • Le message prend la couleur choisie dans le sélecteur de couleur
