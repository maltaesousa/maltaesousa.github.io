OpenLayers - Sélection du fond de plan

Ouvrez cet exemple dans un éditeur de code. Dans le JavaScript vous trouverez 3 constantes contenant chacune un layer.

Exercice 1

Commencez par ajouter une carte à cette page avec un seul layer de votre choix.

Exercice 2

Ajoutez les deux autres layers à la carte mais rendez-les invisibles (propriété "visible" de ol.layer.Tile)

Exercice 3

Créez une fonction JavaScript qui rend invisible toutes les couches puis rend visible une couche dont le nom lui est donné en paramètre.

Exercice 4

Appelez cette fonction depuis la page HTML. La forme est libre, ça peut être trois boutons, des boutons radio, comme vous voulez.

Exercice 5

Vous apprenez que l'HEIG-VD dispose d'un serveur WMS dont voici le lien: http://ogc.heig-vd.ch/. Ajoutez la couche 120307_Yvonand_Plage_25cm disponible à cette adresse.
